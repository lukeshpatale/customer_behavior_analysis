Customer Behavior Data Analytics Project
🔍 Overview

This project focuses on analyzing customer behavior using a complete data analytics workflow. It covers data loading, cleaning, exploratory data analysis (EDA), SQL querying, and building an interactive dashboard. The goal is to extract meaningful insights that help understand customer preferences, sales performance, and business trends.

📁 Dataset

The dataset contains customer-related information such as:

Customer demographics (age, gender)
Product categories (Clothing, Accessories, Footwear, Outerwear)
Purchase details and revenue
Review ratings
Shipping types and subscription status

Python (Pandas, NumPy, Matplotlib, Seaborn) – Data cleaning & EDA
PostgreSQL – Data storage and SQL analysis
Power BI – Interactive dashboard creation

Jupyter Notebook – Development environment
